# Visualizador-de-Quadros-RA
Projeto desenvolvido na disciplina Fundamentos de Realidade Aumentada. Consiste em um visualizador de quadros 3D com AR.js e A‑Frame, permitindo interação via marcadores e botões para trocar molduras, ajustar tamanho e girar os modelos. Compatível com dispositivos móveis e câmera traseira, ideal para explorar conceitos básicos de RA na web.
